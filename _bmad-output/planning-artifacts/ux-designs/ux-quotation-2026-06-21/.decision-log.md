## Designer improvements (Maiga Tech Lab brief: chaleureux / local / modern / reference-worthy)

**Auto-applied (corrections — clearly right):**
- D1 — **Brand unification.** Kill leftover cold near-black `#1c1917` (dark hero, login tile, "TS" placeholder). All dark blocks → brand navy `#1B3070`. Real Q mark everywhere. Single ink = warm navy.
- D2 — **Accessibility floor (WCAG AA).** Amber `#F6A624` never used for text/small UI (fails contrast). Amber = FAB icon, hero glow, logo dot, large-numeral-on-tint only. Text-on-amber-tint → deep `#9a6a07`. All interactive text/links/active states → navy `#1B3070`. Focus ring navy, ≥3:1.
- D3 — **Money is the signature.** Every FCFA amount: Spectral + `tabular-nums`. Consistent across hero, cards, wizard, PDF.
- D4 — **Elevation scale codified.** flat (cards `0 1px 2px`), raised (primary/FAB warm-navy shadow), overlay (sheet/PDF `0 8px 30px`). No ad-hoc shadows.

**Pending user pick:**
- D5 — Local texture (Sahel geometric motif on dark hero + PDF header band).
- D6 — Hero warmth direction.

**Resolved (user picks 2026-06-21):**
- D5 — **Subtle local texture.** Sahel/Tuareg geometric line motif at ~4% opacity (navy-on-navy) on dark hero block + PDF header band only. Filigrane: visible up close, invisible at distance. Not on accent cards. Honors `prefers-reduced-motion` (static).
- D6 — **Navy + sunset glow.** Hero base = deep brand navy `#1B3070`; radial glow top-right = amber `#F6A624` → terracotta `#b8502d`, low opacity, breathing (`glowPulse` 5s). Chaleur via lueur; brand base preserved.
