## Statut du projet

**Phase actuelle : Spécification**

Le projet est en phase de définition. Aucun code n'existe encore. Le cahier des charges définit :

1. **Vision** : Application web de gestion de devis logistiques pour le Niger, puis extension AES
2. **Utilisateurs cibles** : Transporteurs, transitaires, commissionnaires, négociants
3. **Fonctionnalités MVP** : Auth, paramètres société, clients, devis, calculs automatiques, PDF
4. **Stack recommandée** : Next.js/React + Tailwind + PostgreSQL/Supabase
5. **Roadmap** : 3 phases (MVP Niger → Consolidation → Extension AES)
